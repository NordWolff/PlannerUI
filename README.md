# Worklifeplaner MVP

Projekt-Prototyp mit Vue 3 Frontend und Node.js Backend.

## Struktur

- `backend/` - Express API mit In-Memory-Daten
- `frontend/` - Vite + Vue 3 App

## Start

1. `npm install`
2. `npm run dev`

Das startet:
- Backend: http://localhost:4000
- Frontend: http://localhost:5173

## Features

- Benutzerregistrierung und Login
- Teams verwalten
- Projekte erstellen
- Ticket- und Kanban-Ansicht
- Dark/Light Mode
- In-Memory-Daten für MVP-Betrieb
